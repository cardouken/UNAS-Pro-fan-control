# UniFi UNAS Integration

Monitor and control your Ubiquiti UniFi UNAS from Home Assistant with comprehensive drive monitoring, system metrics, and automatic fan control.

## Features

✅ **Automatic Setup** - Installs monitoring scripts via SSH  
✅ **Complete SMART Data** - Temperature, health, power-on hours for all drives  
✅ **System Monitoring** - CPU, memory, storage pool metrics  
✅ **Fan Control** - Automatic temperature-based fan curves  
✅ **Auto-Recovery** - Reinstalls scripts after firmware updates  
✅ **MQTT Integration** - All sensors via auto-discovery  

## Quick Start

1. Install via HACS
2. Add Integration in HA UI
3. Enter UNAS IP and SSH credentials
4. Done! All sensors appear automatically

## Requirements

- UniFi UNAS with SSH enabled
- Root access credentials
- MQTT broker (Mosquitto addon) in Home Assistant
